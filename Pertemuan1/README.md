📂 Pertemuan 1: Dasar Logika & Klasifikasi Mood
Repositori ini berisi tugas pertama mata kuliah Praktikum Kecerdasan Buatan yang berfokus pada implementasi logika dasar menggunakan bahasa pemrograman Python.

📝 Deskripsi Tugas
Program mood_program.py dirancang untuk melakukan klasifikasi kondisi emosional (mood) berdasarkan input pengguna dengan menerapkan struktur kendali logika.

📂 Struktur Folder
Pertemuan1/
├── mood_program.py    # File utama program logika mood
└── README.md          # Dokumentasi tugas pertemuan 1

🛠️ Variabel yang Digunakan
Dalam program ini, digunakan beberapa variabel utama untuk memproses logika:
- input_user: Menyimpan data mentah yang dimasukkan oleh pengguna.
- mood_score: Variabel numerik untuk menentukan bobot kondisi emosional.
- status: Output string yang menampilkan hasil klasifikasi akhir.

🚀 Cara Menjalankan
1. Pastikan Python sudah terinstal di perangkat kamu.
2. Buka terminal atau Command Prompt.
3. Masuk ke direktori folder: Bashcd Pertemuan1
4. Jalankan program dengan perintah: Bashpython mood_program.py
